# T-012 Energy Drink Landing Pages

Built three professional static HTML product-detail / landing pages for the Amazon Associate energy-drink niche with a summer nightlife and EDM-festival tie-in:

- `pages/neon-surge/index.html`
- `pages/afterglow-hydra/index.html`
- `pages/nightdrive-pulse/index.html`

Each page includes a generated premium energy-drink hero/product visual, strong hero copy, product story, key benefits, ingredient/spec highlights, occasions blocks, social-proof style sections, and a clear CTA.

Affiliate links are intentionally placeholders with `href="#"` under the traffic-first rule. Real Amazon Associate URLs can be added later after traffic qualifies.

Generated visual assets are stored in `pages/assets/` and referenced locally by the HTML pages.
