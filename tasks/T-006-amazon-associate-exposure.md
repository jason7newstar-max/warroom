# T-006 — Amazon Associate Program 曝光度提升建议

## 目标
提高 Amazon Associate Program 的有效曝光：不是单纯堆链接，而是增加合规可持续的流量、点击率、转化率和复购入口。

## 先守住合规底线
- 每个含联盟链接的页面/帖子必须有清晰的 affiliate disclosure，放在用户点击链接前能看到的位置。
- 只用 Amazon 允许的 Special Links / SiteStripe / PA API / Creators API 等官方方式生成链接和展示 Program Content；不要截图、下载、改造 Amazon 商品图后自行托管。
- 不要在广告投放里直接用 Amazon 联盟链接做落地页，也不要隐藏、伪装或误导链接目的地。
- 价格、折扣、库存这类信息容易过期；如果展示，必须有自动更新能力或写明时间，避免误导。
- 参考源：Amazon Associates Operating Agreement https://affiliate-program.amazon.com/help/operating/agreement/ ，Program Policies https://affiliate-program.amazon.com/help/operating/policies ，SiteStripe Help https://affiliate-program.amazon.com/help/topic/stripe.html/

## 曝光增长打法

### 1. 内容入口：做“购买意图”而不是泛流量
优先做这四类页面：
- Best X for Y：例如 “best office chair for small apartment”
- X vs Y：对比型关键词，转化意图强
- Alternatives / accessories：替代品、配件、耗材、补充购买
- Problem-solution：从具体痛点切入，再推荐商品

每篇内容固定结构：
- 一句话结论
- 适用人群
- 3-5 个推荐项
- 为什么选它
- 不适合谁
- Amazon CTA

### 2. SEO：按主题集群做，不要散写
选 2-3 个垂直主题先打穿，例如 home office、kitchen gadgets、travel essentials。每个主题做：
- 1 篇 pillar page
- 8-12 篇长尾购买意图文章
- 每篇互链回 pillar page
- 每月更新销量、替代品、已下架链接

目标是让 Google 认为站点在某个小类目有持续权威，而不是杂货铺。

### 3. 社媒：把内容拆成可重复分发的短格式
每篇文章拆成：
- TikTok / Reels：15-30 秒 “3 picks under $X”
- Pinterest：商品清单图、对比图、场景图
- YouTube Shorts：before/after 或 quick test
- X/Threads：短清单 + 原文链接

注意：社媒链接要符合 Amazon 对站点/社媒账号申报和 disclosure 的要求。能导回自有页面时，优先导回自有页面，页面里再放 Amazon 链接。

### 4. 转化：让链接位置更自然
每篇文章至少有三个入口：
- 首屏结论区：Best overall / Best budget / Best premium
- 正文推荐卡片：每个商品一个 CTA
- 结尾购买清单：汇总所有 Amazon 链接

CTA 文案不要写“支持我们”这类弱动机，改成购买意图强的：
- Check current price
- See color options
- Compare sizes
- View on Amazon

### 5. 邮件/私域：把一次性访问变成重复曝光
做一个轻量 newsletter：
- 每周 1 封：本周 5 个值得买
- 节日/Prime Day/Black Friday 前：专题清单
- 新文章发布：只发摘要 + 入口

邮件里也要有 affiliate disclosure；如果合规不确定，邮件优先导回自有页面。

## 30 天执行计划
- 第 1 周：选定 1 个垂直主题；列 30 个关键词；审计现有链接和 disclosure。
- 第 2 周：发布 1 篇 pillar page + 4 篇长尾购买意图文章。
- 第 3 周：每篇文章拆 3 条短视频/图片内容，分发到 TikTok、Pinterest、YouTube Shorts。
- 第 4 周：根据 Amazon Associates 报表和站点分析，按点击率、转化率、收入/千访问排序，更新标题、CTA 和推荐顺序。

## 核心指标
- Impressions：SEO 展示、社媒展示
- CTR：页面到 Amazon 的点击率
- Conversion rate：Amazon 点击后的购买转化
- EPC：每次点击收益
- RPM：每千页面访问收益
- Link health：失效/下架/不合规链接数量

## Dali 建议
先不要扩大到所有类目。用一个主题做 30 天小闭环，目标是验证 “内容集群 + 短内容分发 + 页面 CTA 优化” 是否能把 Amazon 点击率和 EPC 拉起来。第一阶段只追两个结果：Amazon outbound CTR 提升、EPC 提升；如果只涨曝光不涨 EPC，说明流量不够购买意图，需要回到关键词和内容角度重做。
