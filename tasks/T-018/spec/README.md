# T-018 Spec Preview Artifacts

Private cold-pitch spec pages for two emerging wellness / energy brands.

## Brands

- `gorgie/` — GORGIE Protein Energy Drink, Power Punch.
- `bloom/` — Bloom Nutrition Sparkling Energy, Summer Splash.

Each brand folder includes:

- `source/` — real photos curled from the brand's site / Shopify CDN.
- `assets/` — local cinematic makeovers preserving the original product and packaging.
- `index.html` — one-screen spec preview page.
- `vercel.json` — static Vercel-ready config.
