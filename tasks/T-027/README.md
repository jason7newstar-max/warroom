# T-027 ONE TEN LAB Client Micro-Site Template

Single-file HTML template: `template/index.html`

The file has no external CSS or JavaScript dependencies. Replace the `{{TOKEN}}` placeholders with each client's copy before publishing.

## Core tokens

- `{{BRAND}}` - client business name.
- `{{CATEGORY}}` - short business category, such as "Neighborhood Cafe" or "Boutique Studio".
- `{{TAGLINE}}` - hero one-sentence value proposition.
- `{{CTA_EMAIL}}` - email address used in all mailto calls to action.
- `{{CTA_LABEL}}` - primary hero button text.
- `{{SECONDARY_CTA}}` - secondary hero button text.

## Section tokens

- `{{ABOUT_HEADLINE}}`, `{{ABOUT}}`, `{{HIGHLIGHT_STAT}}`, `{{HIGHLIGHT_TEXT}}` - about section copy.
- `{{SERVICES_HEADLINE}}` - menu/services section headline.
- `{{SERVICE_1_LABEL}}`, `{{SERVICE_1_NAME}}`, `{{SERVICE_1_DESC}}`, `{{SERVICE_1_PRICE}}` - first service/menu item.
- `{{SERVICE_2_LABEL}}`, `{{SERVICE_2_NAME}}`, `{{SERVICE_2_DESC}}`, `{{SERVICE_2_PRICE}}` - second service/menu item.
- `{{SERVICE_3_LABEL}}`, `{{SERVICE_3_NAME}}`, `{{SERVICE_3_DESC}}`, `{{SERVICE_3_PRICE}}` - third service/menu item.
- `{{GALLERY_HEADLINE}}`, `{{GALLERY_1_TITLE}}`, `{{GALLERY_1_CAPTION}}`, `{{GALLERY_1_ALT}}`, and matching `2` / `3` tokens - three gallery placeholders.
- `{{CONTACT_HEADLINE}}`, `{{CONTACT_COPY}}`, `{{ADDRESS}}`, `{{HOURS}}`, `{{CONTACT_BUTTON}}` - contact section details.
- `{{FOOTER_NOTE}}` - footer note, such as neighborhood, year, or studio credit.
