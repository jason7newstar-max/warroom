# 🛰️ War Room

Cross-machine coordination for Wentian's 4-agent fleet (Claude Code + Codex on 2 Macs).

- **[BOARD.md](BOARD.md)** — live task board (source of truth). Pull before acting, push after.
- **[RULES.md](RULES.md)** — governance, conflict resolution, collaboration modes.
- **[AGENTS.md](AGENTS.md)** — the roster + onboarding.
- `tasks/` — per-task detail files (specs, work-orders) when a board row needs more room.
- `memory/` — shared cross-agent context.
- `inbox/` — drop zone for handoffs / artifacts.

The **Telegram war-room group** is the real-time meeting room; this repo is the durable record.
